ARRAYS-INSERTION SORT
Take as input N, the size of array. 
Take N more inputs and store that in an array. 
Write a function that insertion sorts the array. Print the elements of sorted array.

Input Format:
Constraints:
Length of Array should be between 1 and 1000.

Output Format
Sample Input
4
3
4
2
1
Sample Output
1
2
3
4
